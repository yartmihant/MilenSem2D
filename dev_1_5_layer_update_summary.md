# Глава V: Переработка структуры слоев модели

## Краткий реферат

В данной главе выполнена полная переработка структуры слоев геологической модели на основе новых данных каротажных скважин. Реализован парсинг 75 геологических слоев из Excel файла, построение геометрии границ слоев с использованием квадратичной интерполяции, а также генерация всех необходимых файлов для построения сетки в Fidesys.

## Задачи главы

### Задача 1: Парсинг геологических слоев
Анализ Excel файла с каротажными данными и извлечение информации о границах слоев для обеих скважин.

### Задача 2: Построение геометрии слоев
Создание геометрии границ слоев между скважинами с использованием квадратичного сплайна и фиксированных Z-координат сейсмических поверхностей.

### Задача 3: Генерация FDS модели
Создание JOU-скриптов для построения геометрии и сетки модели в Fidesys.

## Входные данные

### Геологические данные
- **`src/wells_geometry_sketch_corr_.xlsx`** - Excel файл с каротажными данными
  - Границы слоев выделены жирным шрифтом
  - Названия формаций для обеих скважин
  - Глубины границ слоев

### Сейсмические данные
- **`data/dev_1_2_profile_sections_depths.npz`** - профильные сечения сейсмических поверхностей
  - 11 сейсмических поверхностей
  - Координаты вдоль профиля (1176 точек)
  - Глубины поверхностей

## Методы обработки

### 1. Парсинг геологических слоев

**Функция:** `parse_geological_layers()`

**Алгоритм:**
1. Загрузка Excel файла с помощью openpyxl
2. Поиск ячеек с жирным шрифтом (маркируют границы слоев)
3. Извлечение названий формаций и глубин границ для каждой скважины
4. Создание пар соответствующих слоев (скважина 1 ↔ скважина 2)
5. Сортировка слоев по глубине

**Результат:** 75 геологических слоев с названиями формаций и глубинами границ в обеих скважинах.

### 2. Конвертация координат

**Функции:** `local_to_global_depth()` и `global_to_local_depth()`

Система координат:
- **Глобальные координаты (D):** реальные глубины в метрах
- **Локальные координаты (Z):** параметризованные координаты между сейсмическими поверхностями

**Алгоритм:**
- Интерполяция между фиксированными Z-координатами: [9, 18, 34, 42, 49, 89, 138, 167, 230, 235, 249]
- Обратная и прямая конвертация с учетом локальной геометрии сейсмических поверхностей

### 3. Построение границ слоев

**Функция:** `build_layer_boundary_line()`

**Алгоритм:**
1. Определение локальных Z-координат границ слоев в скважинах
2. Построение квадратичного сплайна по четырем опорным точкам:
   - (0, Z₁) - начало профиля
   - (4250, Z₁) - скважина 1
   - (7500, Z₂) - скважина 2
   - (11750, Z₂) - конец профиля
3. Конвертация локальных координат обратно в глобальные глубины
4. Коррекция минимальной толщины слоя до 10 м

**Параметры:**
- Скважина 1: X = 4250 м от начала профиля
- Скважина 2: X = 7500 м от начала профиля
- Длина профиля: 11750 м
- Минимальная толщина слоя: 10 м

### 4. Генерация JOU-скриптов для Fidesys

**Скрипты геометрии:**
- **`data/dev_1_5_model_vertex.jou`** - создание вершин для границ слоев
  - 76 линий (поверхность + 75 границ слоев)
  - 236 точек на линию (шаг 50 м)
  
- **`data/dev_1_5_model_spline.jou`** - создание сплайновых кривых и поверхностей
  - Разделение на 3 части: левая (до скважины 1), центральная, правая (после скважины 2)
  - Генерация поверхностей методом skin curve

**Скрипты сетки:**
- **`data/dev_1_5_model_mesh_curve.jou`** - параметры дискретизации кривых
  - Горизонтальные группы: 425, 325, 425 интервалов
  - Вертикальные группы: адаптивные по слоям
  
- **`data/dev_1_5_model_mesh_surface.jou`** - построение сетки поверхностей
  - Метод map для структурированных областей
  - Метод pave для неструктурированных областей

## Использование результатов

### Загрузка данных слоев
```python
import numpy as np

# Загрузка границ слоев
layer_data = np.load('data/dev_1_5_2_layer_boundaries_quadratic.npz')
layer_boundaries_array = layer_data['layer_boundaries_array']  # (75, 1176)
formations = layer_data['formations']  # (75,)
well1_depths = layer_data['well1_depths']  # (75,)
well2_depths = layer_data['well2_depths']  # (75,)
distances = layer_data['distances']  # (1176,)

print(f"Количество слоев: {len(formations)}")
print(f"Длина профиля: {distances[-1]} м")
```

### Работа в Fidesys
```bash
# Последовательное выполнение скриптов в Fidesys:
# 1. Создание геометрии
playback 'data/dev_1_5_model_vertex.jou'
playback 'data/dev_1_5_model_spline.jou'

# 2. Построение сетки
playback 'data/dev_1_5_model_mesh_curve.jou'
playback 'data/dev_1_5_model_mesh_surface.jou'

# 3. Сохранение модели
export fds 'data/dev_1_5_model.fds'
```
