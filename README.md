# MILEN SEM 2D: Моделирование сейсмических колебаний

**Проект:** Численное моделирование распространения сейсмических волн в двумерной среде методом спектральных элементов

**Цель:** Создание высокоточной конечно-элементной модели с адаптивной сеткой для численного моделирования сейсмических колебаний в сложной слоистой геологической среде

## Использование результатов

### Загрузка данных материалов

```python
import numpy as np

# Декартова сетка (для конечных разностей)
data = np.load('data/dev_1_7_material_grids.npz')
material_grid = data['material_grid']  # [2350, 550, 3] (E, ν, ρ)
coords_grid = data['coords_grid']      # [2350, 550, 2] (x, y)
layer_indexes = data['layer_indexes_grid']  # [2350, 550]

# Расчет скоростей волн
E = material_grid[:, :, 0]
nu = material_grid[:, :, 1]
rho = material_grid[:, :, 2]

Vp = np.sqrt(E / rho * (1 - nu) / ((1 + nu) * (1 - 2*nu)))
Vs = np.sqrt(E / (2 * rho * (1 + nu)))
```

### Загрузка SEG-Y файлов (Tesseral)

```python
import segyio

    print(f"Диапазон Vp: {vp_data.min():.1f} - {vp_data.max():.1f} м/с")
```
